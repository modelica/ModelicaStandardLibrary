.ModelicaReference.Operators.'abs()'

Information

Absolute value of Real or Integer variable.

Syntax

abs(v)

Description

Is expanded into "noEvent(if v ≥ 0 then v else -v)". Argument v needs to be an Integer or Real expression.

Examples

  abs({-3, 0, 3})
 = {3, 0, 3}
