water_ph

Calculate the property record for dynamic simulation properties using p,h as states

Syntax

pro = water_ph(p, h, phase)

Inputs (3)

p

Type: Pressure (Pa)

Description: Pressure
h

Type: SpecificEnthalpy (J/kg)

Description: Specific enthalpy
phase

Default Value: 0

Type: Integer

Description: Phase: 2 for two-phase, 1 for one phase, 0 if unknown

Outputs (1)

pro

Type: ThermoProperties_ph

Description: Property record for dynamic simulation
