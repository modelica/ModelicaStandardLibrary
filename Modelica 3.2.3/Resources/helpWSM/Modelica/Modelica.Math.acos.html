acos

Inverse cosine (-1 <= u <= 1)

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

This function returns y = acos(u), with -1 ≤ u ≤ +1:

Syntax

y = acos(u)

Inputs (1)

u

Type: Real

Outputs (1)

y

Type: Angle (rad)
