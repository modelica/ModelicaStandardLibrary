findInterval

Find interval for the root

Syntax

(alpha_min, alpha_max) = findInterval(c1, c2)

Inputs (2)

c1

Type: Real[:]

Description: [p] coefficients of denominator polynomials (a*p + 1)
c2

Type: Real[:,2]

Description: [p^2, p] coefficients of denominator polynomials (b*p^2 + a*p + 1)

Outputs (2)

alpha_min

Type: Real
alpha_max

Type: Real
