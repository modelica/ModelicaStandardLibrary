massFlowRate_dp

Return mass flow rate from constant loss factor data and pressure drop (m_flow = f(dp))

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

Compute mass flow rate from constant loss factor and pressure drop (m_flow = f(dp)). For small pressure drops (dp < dp_small), the characteristic is approximated by a polynomial in order to have a finite derivative at zero mass flow rate.

Syntax

m_flow = massFlowRate_dp(dp, rho_a, rho_b, data, dp_small)

Inputs (5)

dp

Type: Pressure (Pa)

Description: Pressure drop (dp = port_a.p - port_b.p)
rho_a

Type: Density (kg/m³)

Description: Density at port_a
rho_b

Type: Density (kg/m³)

Description: Density at port_b
data

Type: LossFactorData

Description: Constant loss factors for both flow directions
dp_small

Default Value: 1

Type: AbsolutePressure (Pa)

Description: Turbulent flow if |dp| >= dp_small

Outputs (1)

m_flow

Type: MassFlowRate (kg/s)

Description: Mass flow rate from port_a to port_b
