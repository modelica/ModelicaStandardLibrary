junction2SPICE3MOSFETRevised

Junction current and conductance calculation

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

This internal function calculates both the junction current and the junction conductance dependent from the given voltage.

Syntax

(out_current, out_cond) = junction2SPICE3MOSFETRevised(voltage, temp, ncoeff, satcur)

Inputs (4)

voltage

Type: Voltage (V)

Description: Input voltage
temp

Type: Temp_K (K)

Description: Device Temperature
ncoeff

Type: Real
satcur

Type: Current (A)

Description: Saturation current

Outputs (2)

out_current

Type: Current (A)

Description: Calculated current
out_cond

Type: Conductance (S)

Description: Calculated conductance
