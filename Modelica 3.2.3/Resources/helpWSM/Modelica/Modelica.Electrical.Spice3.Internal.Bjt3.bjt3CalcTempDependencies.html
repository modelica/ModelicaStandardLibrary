bjt3CalcTempDependencies

Temperature dependency calculation, obsolete

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

In this function for the bipolar transistor model temperature dependencies are calculated using temperature treating functions from the equation package.

Syntax

out_c = bjt3CalcTempDependencies(in_p3, in_p, m, in_vl)

in_p3

Description: Input record Bjt3
in_p

Description: Input record with Bjt model line parameters
m

Type: Model

Description: Input record model
