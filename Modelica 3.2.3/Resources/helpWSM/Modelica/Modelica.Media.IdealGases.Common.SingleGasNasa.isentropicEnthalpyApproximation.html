isentropicEnthalpyApproximation

Approximate method of calculating h_is from upstream properties and downstream pressure

Syntax

h_is = isentropicEnthalpyApproximation(p2, state, exclEnthForm, refChoice, h_off)

Inputs (5)

p2

Type: Pressure (Pa)

Description: Downstream pressure
state

Type: ThermodynamicState

Description: Properties at upstream location
exclEnthForm

Default Value: Functions.excludeEnthalpyOfFormation

Type: Boolean

Description: If true, enthalpy of formation Hf is not included in specific enthalpy h
refChoice

Default Value: Functions.referenceChoice

Type: ReferenceEnthalpy

Description: Choice of reference enthalpy
h_off

Default Value: Functions.h_offset

Type: SpecificEnthalpy (J/kg)

Description: User defined offset for reference enthalpy, if referenceChoice = UserDefined

Outputs (1)

h_is

Type: SpecificEnthalpy (J/kg)

Description: Isentropic enthalpy
