diodeNoBypassCode

Calculation of currents

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

This function NoBypassCode calculates the currents (and the capacitances) that are necessary for the currents to be used in the top-level model.

Syntax

(out_cc, m_dCap) = diodeNoBypassCode(in_p, in_dp, in_c, in_m, in_m_mbInit, in_m_pVoltageValues)

Inputs (6)

in_p

Type: DiodeModelLineParams

Description: Input record model line parameters for diode
in_dp

Type: DiodeParams

Description: Input record with parameters for diode
in_c

Type: DiodeCalc

Description: Input record DiodeCalc
in_m

Type: Model

Description: Input record model
in_m_mbInit

Type: Boolean
in_m_pVoltageValues

Type: Voltage[2] (V)

Outputs (2)

out_cc

Type: CurrentsCapacitances

Description: Output record with calculated currents and capacitances
m_dCap

Type: Capacitance (F)

Description: Output capacitance
