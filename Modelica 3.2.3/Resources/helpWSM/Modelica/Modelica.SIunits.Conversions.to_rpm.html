to_rpm

Convert from radian per second to revolutions per minute

Syntax

rpm = to_rpm(rs)

Inputs (1)

rs

Type: AngularVelocity (rad/s)

Description: radian per second value

Outputs (1)

rpm

Type: AngularVelocity_rpm (rev/min)

Description: revolutions per minute value
