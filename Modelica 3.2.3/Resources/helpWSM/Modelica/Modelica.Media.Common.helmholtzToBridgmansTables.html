helmholtzToBridgmansTables

Calculates base coefficients for Bridgmans tables from Helmholtz energy

Syntax

(v, p, T, s, cp, alpha, gamma) = helmholtzToBridgmansTables(f)

Inputs (1)

f

Type: HelmholtzDerivs

Description: Dimensionless derivatives of Helmholtz function

Outputs (7)

v

Default Value: 1 / f.d

Type: SpecificVolume (m³/kg)

Description: Specific volume
p

Type: Pressure (Pa)

Description: Pressure
T

Default Value: f.T

Type: Temperature (K)

Description: Temperature
s

Type: SpecificEntropy (J/(kg·K))

Description: Specific entropy
cp

Type: SpecificHeatCapacity (J/(kg·K))

Description: Heat capacity at constant pressure
alpha

Type: IsobaricVolumeExpansionCoefficient (¹/K)

Description: Isobaric volume expansion coefficient
gamma

Type: IsothermalCompressibility (¹/Pa)

Description: Isothermal compressibility
