to_kmh

Convert from metre per second to kilometre per hour

Syntax

kmh = to_kmh(ms)

Inputs (1)

ms

Type: Velocity (m/s)

Description: metre per second value

Outputs (1)

kmh

Type: Velocity_kmh (km/h)

Description: kilometre per hour value
