massToMoleFractions

Return mole fractions from mass fractions X

Syntax

moleFractions = massToMoleFractions(X, MMX)

Inputs (2)

X

Type: MassFraction[:]

Description: Mass fractions of mixture
MMX

Type: MolarMass[:] (kg/mol)

Description: Molar masses of components

Outputs (1)

moleFractions

Type: MoleFraction[size(X, 1)]

Description: Mole fractions of gas mixture
