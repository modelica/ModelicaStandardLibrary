brushVoltageDrop

Voltage drop of carbon brushes

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

Calculates the voltage drop of carbon brushes, according to Brush losses, e.g., used for initial equations.

Syntax

v = brushVoltageDrop(brushParameters, i)

Inputs (2)

brushParameters

Type: BrushParameters

Description: Brush loss parameters
i

Type: Current (A)

Description: Actual current

Outputs (1)

v

Type: Voltage (V)

Description: Voltage drop
