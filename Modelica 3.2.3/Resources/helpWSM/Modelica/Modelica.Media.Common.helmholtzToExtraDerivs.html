helmholtzToExtraDerivs

Compute additional thermodynamic derivatives from dimensionless Helmholtz function

Syntax

dpro = helmholtzToExtraDerivs(f)

Inputs (1)

f

Type: HelmholtzDerivs

Description: Dimensionless derivatives of Helmholtz function

Outputs (1)

dpro

Type: ExtraDerivatives

Description: Additional property derivatives
