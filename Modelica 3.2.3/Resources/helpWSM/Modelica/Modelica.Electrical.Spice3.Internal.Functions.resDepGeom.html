resDepGeom

Resistance dependent from width and narrow

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

This internal function calculates the resistance in dependency from the geometrical values (width, narrow) and resistivity.

Syntax

out = resDepGeom(rsh, width, length, narrow)

Inputs (4)

rsh

Type: Real

Description: Input sheet resistance
width

Type: Length (m)

Description: Input transistor width
length

Type: Length (m)

Description: Input transistor length
narrow

Type: Length (m)

Description: Input narrow

Outputs (1)

out

Type: Real

Description: Output value
