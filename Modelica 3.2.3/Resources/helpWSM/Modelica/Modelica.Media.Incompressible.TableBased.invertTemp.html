invertTemp

Function to invert temperatures

Syntax

invTable = invertTemp(table, Tink)

Inputs (2)

table

Type: Real[:]

Description: Table temperature data
Tink

Type: Boolean

Description: Flag for Celsius or Kelvin

Outputs (1)

invTable

Type: Real[size(table, 1)]

Description: Inverted temperatures
