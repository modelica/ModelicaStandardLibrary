spliceFunction

Spline interpolation of two functions

Syntax

out = spliceFunction(pos, neg, x, deltax)

Inputs (4)

pos

Type: Real

Description: Returned value for x-deltax >= 0
neg

Type: Real

Description: Returned value for x+deltax <= 0
x

Type: Real

Description: Function argument
deltax

Default Value: 1

Type: Real

Description: Region around x with spline interpolation

Outputs (1)

out

Type: Real
