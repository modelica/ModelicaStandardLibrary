Gibbs2_dT

Function to calculate analytic derivatives for computing p given d and T

Syntax

nderivs = Gibbs2_dT(g)

Inputs (1)

g

Type: GibbsDerivs2

Description: Dimensionless derivatives of Gibbs function

Outputs (1)

nderivs

Type: NewtonDerivatives_dT

Description: Derivatives for Newton iteration to compute p from d and T
