extraDerivs_ph

Function to calculate some extra thermophysical properties in regions 1, 2, 3 and 5 as f(p,h)

Syntax

dpro = extraDerivs_ph(p, h, phase)

Inputs (3)

p

Type: Pressure (Pa)

Description: Pressure
h

Type: SpecificEnthalpy (J/kg)

Description: Specific enthalpy
phase

Default Value: 0

Type: Integer

Description: Phase: 2 for two-phase, 1 for one phase, 0 if unknown

Outputs (1)

dpro

Type: ExtraDerivatives

Description: Thermodynamic property collection
