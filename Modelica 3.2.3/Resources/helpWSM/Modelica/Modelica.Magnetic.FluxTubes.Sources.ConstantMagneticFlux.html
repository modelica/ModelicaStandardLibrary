ConstantMagneticFlux

Source of constant magnetic flux

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

Sources of a constant magnetic flux are useful for modelling of permanent magnets with Norton's magnetic equivalent circuit.

Parameters (1)

Phi

Value: 1

Type: MagneticFlux (Wb)

Description: Magnetic flux

Connectors (2)

port_p

Type: PositiveMagneticPort

Description: Positive magnetic port
port_n

Type: NegativeMagneticPort

Description: Negative magnetic port
