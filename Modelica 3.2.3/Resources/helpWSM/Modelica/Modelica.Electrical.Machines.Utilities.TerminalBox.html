TerminalBox

Terminal box Y/D-connection

Diagram

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

TerminalBox: at the bottom connected to both machine plugs, connect at the top to the grid as usual,
choosing Y-connection (StarDelta=Y) or D-connection (StarDelta=D).

Parameters (1)

terminalConnection

Value:

Type: String

Description: Choose Y=star/D=delta

Connectors (4)

plug_sp

Type: PositivePlug

Description: To positive stator plug
plug_sn

Type: NegativePlug

Description: To negative stator plug
plugSupply

Type: PositivePlug

Description: To grid
starpoint

Type: NegativePin

Components (2)

star

Type: Star
delta

Type: Delta
