bjtInitEquations

Initial calculation, obsolete

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

Within this function some parameters are initially precalculated from model line parameters.

Syntax

out_v = bjtInitEquations(in_p, in_pml, in_vl)

Inputs (3)

in_p

Type: Bjt

Description: Input record Bjt
in_pml

Type: BjtModelLineParams

Description: Input record with Bjt model line parameters
in_vl

Type: BjtModelLineVariables

Description: Input record with Bjt model line variables

Outputs (1)

out_v

Type: BjtVariables

Description: Output record with Bjt variables
