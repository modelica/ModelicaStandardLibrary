getNumberOfFiles

Get number of files and directories in a directory (POSIX functions opendir, readdir, closedir)

Syntax

result = getNumberOfFiles(directory)

Inputs (1)

directory

Type: String

Description: Directory name

Outputs (1)

result

Type: Integer

Description: Number of files and directories present in 'directory'
