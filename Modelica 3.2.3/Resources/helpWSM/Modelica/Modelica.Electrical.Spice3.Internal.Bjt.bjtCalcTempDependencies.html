bjtCalcTempDependencies

Temperature dependency calculation

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

In this function for the bipolar transistor model temperature dependencies are calculated using temperature treating functions from the equation package.

Syntax

out_c = bjtCalcTempDependencies(in_p3, in_p)

Inputs (2)

in_p3

Type: Bjt

Description: Input record Bjt
in_p

Type: BjtModelLineParams

Description: Input record with Bjt model line parameters

Outputs (1)

out_c

Type: BjtCalc

Description: Output record BjtCalc
