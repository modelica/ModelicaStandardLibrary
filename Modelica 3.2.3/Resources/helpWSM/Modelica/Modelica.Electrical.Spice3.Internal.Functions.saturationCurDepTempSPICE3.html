saturationCurDepTempSPICE3

Temperature dependency of saturation current

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

This internal function calculates the temperature dependent saturation current based on the actual and the nominal temperature.

Syntax

ret = saturationCurDepTempSPICE3(satcur0, temp, tnom, emissioncoeff, energygap, satcurexp)

Inputs (6)

satcur0

Type: Current (A)

Description: Saturation current
temp

Type: Temp_K (K)

Description: Device Temperature
tnom

Type: Temp_K (K)

Description: Nominal Temperature
emissioncoeff

Type: Real
energygap

Type: Real
satcurexp

Type: Real

Outputs (1)

ret

Type: Current (A)

Description: Output value
