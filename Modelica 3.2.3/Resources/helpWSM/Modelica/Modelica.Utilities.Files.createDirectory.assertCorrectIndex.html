assertCorrectIndex

Print error, if index to last essential character in directory is wrong

Syntax

assertCorrectIndex(index, directoryName)

Inputs (2)

index

Type: Integer

Description: Index must be > 0
directoryName

Type: String

Description: Directory name for error message
