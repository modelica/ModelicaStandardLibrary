partialTruncatedQuantile

Common interface of truncated quantile functions (= inverse cumulative distribution functions)

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

A partial function containing the common arguments of the quantile functions for truncated distributions.

Inputs (3)

u

Type: Real

Description: Random number in the range 0 <= u <= 1
y_min

Default Value: 0

Type: Real

Description: Lower limit of y
y_max

Default Value: 1

Type: Real

Description: Upper limit of y

Outputs (1)

y

Type: Real

Description: Random number u transformed according to the given distribution

quantile
