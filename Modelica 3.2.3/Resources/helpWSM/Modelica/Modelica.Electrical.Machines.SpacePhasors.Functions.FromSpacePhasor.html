FromSpacePhasor

Conversion from space phasor and zero sequence component to multi phase

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

Transformation of space phasor and zero sequence value to multi phase values (of voltages or currents).

Syntax

y = FromSpacePhasor(x, x0, m)

Inputs (3)

x

Type: Real[2]

Description: Space phasor
x0

Type: Real

Description: Zero sequence component
m

Type: Integer

Description: Number of phases

Outputs (1)

y

Type: Real[m]

Description: Multi phase output
