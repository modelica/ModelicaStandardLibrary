flipLeftRight

Flip the columns of a matrix in left/right direction

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

Syntax

         A_flr = Matrices.flipLeftRight(A);

Description

Function flipLeftRight computes from matrix A a matrix A_flr with flipped columns, i.e., A_flr[:,i]=A[:,n-i+1], i=1,..., n.

Example

  A = [1, 2,  3;
       3, 4,  5;
      -1, 2, -3];

  A_flr = flipLeftRight(A);

  results in:

  A_flr = [3, 2,  1;
           5, 4,  3;
          -3, 2, -1]

See also

Matrices.flipUpDown

Syntax

Aflip = flipLeftRight(A)

Inputs (1)

A

Type: Real[:,:]

Description: Matrix to be flipped

Outputs (1)

Aflip

Type: Real[size(A, 1),size(A, 2)]

Description: Flipped matrix
