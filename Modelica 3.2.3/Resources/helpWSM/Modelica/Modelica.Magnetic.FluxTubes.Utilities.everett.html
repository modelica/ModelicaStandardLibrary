everett

Everett function, see [YUY89]

Syntax

J = everett(a_, b_, p, mirror)

Inputs (4)

a_

Type: MagneticFieldStrength (A/m)
b_

Type: MagneticFieldStrength (A/m)
p

Type: BaseData
mirror

Type: Boolean

Description: If true then J(a,b)=-J(b,a) else J(a,b)=0 for a

Outputs (1)

J

Type: MagneticFluxDensity (T)

Description: Magnetic polarisation
