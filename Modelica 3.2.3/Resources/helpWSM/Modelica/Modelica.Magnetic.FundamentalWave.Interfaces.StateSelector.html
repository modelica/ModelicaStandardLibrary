StateSelector

Transform instantaneous values to space phasors and select states

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

Transforms instantaneous values into space phasors and zero system currents, rotates space phasors and sets stateSelect modifiers in order to choose states w.r.t. rotating frame, i.e., with small derivatives.

Parameters (3)

x0StateSelect

Value: StateSelect.prefer

Type: StateSelect

Description: Priority to use zero systems as states
xrStateSelect

Value: StateSelect.prefer

Type: StateSelect

Description: Priority to use space phasors w.r.t. rotating frame as states
np

Value: integer((m - 1) / 2)

Type: Integer

Description: Number of space phasors

Inputs (2)

xi

Type: Real[m]

Description: Instantaneous values
gamma

Type: Angle (rad)

Description: Angle of rotation

Components (2)

xf

Type: Complex[np]

Description: Space phasors w.r.t. fixed frame
xr

Type: Complex[np]

Description: Space phasors w.r.t. rotating frame
