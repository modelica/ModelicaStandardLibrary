setState_px

Return thermodynamic state from pressure and vapour quality

Syntax

state = setState_px(p, x)

Inputs (2)

p

Type: AbsolutePressure (Pa)

Description: Pressure
x

Type: MassFraction (kg/kg)

Description: Vapour quality

Outputs (1)

state

Type: ThermodynamicState

Description: Thermodynamic state record
