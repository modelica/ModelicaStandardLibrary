MIMOs

Multiple Input Multiple Output continuous control block with same number of inputs and outputs

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

Block has a continuous Real input and a continuous Real output signal vector where the signal sizes of the input and output vector are identical.

Parameters (1)

n

Value: 1

Type: Integer

Description: Number of inputs (= number of outputs)

Connectors (2)

u

Type: RealInput[n]

Description: Connector of Real input signals
y

Type: RealOutput[n]

Description: Connector of Real output signals
