isentropicEnthalpy

Return isentropic enthalpy

Syntax

h_is = isentropicEnthalpy(p_downstream, refState, exact)

Inputs (3)

p_downstream

Type: AbsolutePressure (Pa)

Description: Downstream pressure
refState

Type: ThermodynamicState

Description: Reference state for entropy
exact

Default Value: false

Type: Boolean

Description: Flag whether exact or approximate version should be used

Outputs (1)

h_is

Type: SpecificEnthalpy (J/kg)

Description: Isentropic enthalpy
