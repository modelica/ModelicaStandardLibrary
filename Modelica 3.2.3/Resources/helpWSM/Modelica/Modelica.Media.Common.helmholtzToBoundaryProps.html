helmholtzToBoundaryProps

Calculate phase boundary property record from dimensionless Helmholtz function

Syntax

sat = helmholtzToBoundaryProps(f)

Inputs (1)

f

Type: HelmholtzDerivs

Description: Dimensionless derivatives of Helmholtz function

Outputs (1)

sat

Type: PhaseBoundaryProperties

Description: Phase boundary property record
