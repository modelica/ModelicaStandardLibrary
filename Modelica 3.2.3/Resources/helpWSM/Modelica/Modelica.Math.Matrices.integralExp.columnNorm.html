columnNorm

Returns the column norm of a matrix

Syntax

result = columnNorm(A)

Inputs (1)

A

Type: Real[:,:]

Description: Input matrix

Outputs (1)

result

Default Value: 0.0

Type: Real

Description: 1-norm of matrix A
