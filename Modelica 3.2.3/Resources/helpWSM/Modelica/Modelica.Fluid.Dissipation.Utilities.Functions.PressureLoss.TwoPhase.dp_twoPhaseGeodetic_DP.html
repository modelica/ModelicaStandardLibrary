dp_twoPhaseGeodetic_DP

Geodetic pressure loss of straight pipe for two phase flow | calculate pressure loss

Syntax

DP_geo = dp_twoPhaseGeodetic_DP(voidFractionApproach, crossSectionalAveraged, length, phi, rho_g, rho_l, x_flow)

Inputs (7)

voidFractionApproach

Default Value: Dissipation.Utilities.Types.VoidFractionApproach.Homogeneous

Type: VoidFractionApproach

Description: Choice of void fraction approach
crossSectionalAveraged

Default Value: true

Type: Boolean

Description: true == cross sectional averaged void fraction | false == volumetric
length

Default Value: 1

Type: Length (m)

Description: Length in fluid flow direction
phi

Default Value: 0

Type: Angle (rad)

Description: Tilt angle to horizontal
rho_g

Type: Density (kg/m³)

Description: Density of gaseous phase
rho_l

Type: Density (kg/m³)

Description: Density of liquid phase
x_flow

Default Value: 0

Type: Real

Description: Mass flow rate quality

Outputs (1)

DP_geo

Type: Pressure (Pa)

Description: Geodetic pressure loss
