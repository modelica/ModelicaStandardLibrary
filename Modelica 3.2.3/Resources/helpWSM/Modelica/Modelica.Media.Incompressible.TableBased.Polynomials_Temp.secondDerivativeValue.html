secondDerivativeValue

Value of 2nd derivative of polynomial at abscissa value u

Syntax

y = secondDerivativeValue(p, u)

Inputs (2)

p

Type: Real[:]

Description: Polynomial coefficients (p[1] is coefficient of highest power)
u

Type: Real

Description: Abscissa value

Outputs (1)

y

Type: Real

Description: Value of 2nd derivative of polynomial at u
