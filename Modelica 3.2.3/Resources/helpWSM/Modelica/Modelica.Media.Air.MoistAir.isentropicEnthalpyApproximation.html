isentropicEnthalpyApproximation

Approximate calculation of h_is from upstream properties, downstream pressure, gas part only

Syntax

h_is = isentropicEnthalpyApproximation(p2, state)

Inputs (2)

p2

Type: AbsolutePressure (Pa)

Description: Downstream pressure
state

Type: ThermodynamicState

Description: Thermodynamic state at upstream location

Outputs (1)

h_is

Type: SpecificEnthalpy (J/kg)

Description: Isentropic enthalpy
