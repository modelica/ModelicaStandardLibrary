Multiplex4

Multiplexer block for four input connectors

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

The output connector is the concatenation of the four input connectors. Note, that the dimensions of the input connector signals have to be explicitly defined via parameters n1, n2, n3 and n4.

Parameters (4)

n1

Value: 1

Type: Integer

Description: Dimension of input signal connector 1
n2

Value: 1

Type: Integer

Description: Dimension of input signal connector 2
n3

Value: 1

Type: Integer

Description: Dimension of input signal connector 3
n4

Value: 1

Type: Integer

Description: Dimension of input signal connector 4

Connectors (5)

u1

Type: RealInput[n1]

Description: Connector of Real input signals 1
u2

Type: RealInput[n2]

Description: Connector of Real input signals 2
u3

Type: RealInput[n3]

Description: Connector of Real input signals 3
u4

Type: RealInput[n4]

Description: Connector of Real input signals 4
y

Type: RealOutput[n1 + n2 + n3 + n4]

Description: Connector of Real output signals
