mos2CalcInitEquationsRevised

Mosfet initial precalculations (level 2)

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

This function mosCalcInitEquations does the initial precalculation of the MOSFET parameters (level 2).

Syntax

out_c = mos2CalcInitEquationsRevised(in_p, in_m)

Inputs (2)

in_p

Type: Mos2ModelLineParams

Description: Input record Mos2 values
in_m

Type: Mosfet

Description: Input record MOSFET parameters

Outputs (1)

out_c

Type: Mos2Calc

Description: Output record Mos2 calculated values
