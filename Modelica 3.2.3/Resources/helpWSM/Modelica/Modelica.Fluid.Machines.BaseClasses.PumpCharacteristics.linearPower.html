linearPower

Linear power consumption characteristic

Syntax

consumption = linearPower(V_flow, V_flow_nominal, W_nominal)

Inputs (3)

V_flow

Type: VolumeFlowRate (m³/s)

Description: Volumetric flow rate
V_flow_nominal

Type: VolumeFlowRate[2] (m³/s)

Description: Volume flow rate for two operating points (single pump)
W_nominal

Type: Power[2] (W)

Description: Power consumption for two operating points

Outputs (1)

consumption

Type: Power (W)

Description: Power consumption
