ToSpacePhasor

Conversion from multi phase input to space phasor and zero sequence component

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

Transformation of multi phase values (of voltages or currents) to space phasor and zero sequence value.

Syntax

(y, y0) = ToSpacePhasor(x)

Inputs (1)

x

Type: Real[:]

Description: Multi phase (voltage or current) input

Outputs (2)

y

Type: Real[2]

Description: Space phasor
y0

Type: Real

Description: Zero sequence component (of voltage or current)
