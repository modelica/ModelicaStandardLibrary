region_pT

Return the current region (valid values: 1,2,3,5) in IF97, given pressure and temperature

Syntax

region = region_pT(p, T, mode)

Inputs (3)

p

Type: Pressure (Pa)

Description: Pressure
T

Type: Temperature (K)

Description: Temperature (K)
mode

Default Value: 0

Type: Integer

Description: Mode: 0 means check, otherwise assume region=mode

Outputs (1)

region

Type: Integer

Description: Region (valid values: 1,2,3,5) in IF97, region 4 is impossible!
