PartialFixedShape

Base class for flux tubes with fixed shape during simulation; linear or non-linear material characteristics

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

Please refer to the description of the subpackage Shapes.FixedShape for utilisation of this partial model.

Parameters (1)

mu_rConst

Value: 1

Type: RelativePermeability

Description: Constant relative permeability

Connectors (2)

port_p

port_n

Type: NegativeMagneticPort

Description: Negative quasi-static magnetic port

Components (4)

V_m

Description: Magnetic field strength (normal component)
