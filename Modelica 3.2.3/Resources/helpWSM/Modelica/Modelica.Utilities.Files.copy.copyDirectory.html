copyDirectory

Copy a directory

Syntax

copyDirectory(oldName, newName, replace)

Inputs (3)

oldName

Type: String

Description: Old directory name without trailing '/'; existence is guaranteed
newName

Type: String

Description: New directory name without trailing '/'; directory was already created
replace

Type: Boolean

Description: = true, if an existing newName may be replaced
