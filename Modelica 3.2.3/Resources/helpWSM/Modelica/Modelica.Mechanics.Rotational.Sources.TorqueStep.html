TorqueStep

Constant torque, not dependent on speed

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

Model of a torque step at time startTime. Positive torque accelerates in positive direction of flange rotation.

Parameters (4)

useSupport

Value: false

Type: Boolean

Description: = true, if support flange enabled, otherwise implicitly grounded
stepTorque

Value:

Type: Torque (N·m)

Description: Height of torque step (if negative, torque is acting as load)
offsetTorque

Value:

Type: Torque (N·m)

Description: Offset of torque
startTime

Value: 0

Type: Time (s)

Description: Torque = offset for time < startTime

Connectors (2)

flange

Type: Flange_b

Description: Flange of shaft
support

Type: Support

Description: Support/housing of component
