thermalConductivity

Compute lambda(d,T,p) in the one-phase region

Syntax

lambda = thermalConductivity(d, T, p, phase, industrialMethod)

Inputs (5)

d

Type: Density (kg/m³)

Description: Density
T

Type: Temperature (K)

Description: Temperature (K)
p

Type: Pressure (Pa)

Description: Pressure
phase

Default Value: 0

Type: Integer

Description: 2 for two-phase, 1 for one-phase, 0 if not known
industrialMethod

Default Value: true

Type: Boolean

Description: If true, the industrial method is used, otherwise the scientific one

Outputs (1)

lambda

Type: ThermalConductivity (W/(m·K))

Description: Thermal conductivity
