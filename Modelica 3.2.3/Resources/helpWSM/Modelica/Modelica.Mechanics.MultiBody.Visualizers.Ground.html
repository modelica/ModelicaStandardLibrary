Ground

Visualizing the ground (box in z=0)

Diagram

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

This shape visualizes the x-y plane by a box.

Parameters (4)

animation

Value: true

Type: Boolean

Description: = true, if animation of ground shall be enabled
length

Value: 10

Type: Position (m)

Description: Length and width of box (center is at x=y=0)
height

Value: 0.02

Type: Position (m)

Description: Height of box (upper surface is at z=0, lower surface is at z=-height)
groundColor

Value: {0, 255, 0}

Type: Color

Description: Color of box

Components (2)

ground

Type: FixedShape
fixed

Type: Fixed
