getMemory

Get Memory

Syntax

(m, data, bit) = getMemory(filename, n_addr, n_data)

Inputs (3)

filename

Type: String
n_addr

Type: Integer

Description: addr width
n_data

Type: Integer

Description: data width

Outputs (3)

m

Type: Logic[integer(2 ^ n_addr),n_data]

Description: Memory with data, lowest bit on left side
data

Type: String
bit

Type: Integer
