diodeInitEquations

Initial calculation

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

In this function some initial calculations for the diode model are done, especially concerning the handling of the breakthrough voltage.

Syntax

out_v = diodeInitEquations(in_p)

Inputs (1)

in_p

Type: DiodeModelLineParams

Description: Input record with diode model line parameters

Outputs (1)

out_v

Type: DiodeVariables

Description: Output record with diode variables
