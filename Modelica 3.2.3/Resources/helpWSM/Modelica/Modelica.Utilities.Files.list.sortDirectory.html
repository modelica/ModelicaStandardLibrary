sortDirectory

Sort directory in directories and files with alphabetic order

Syntax

(orderedNames, nDirectories) = sortDirectory(directory, names)

Inputs (2)

directory

Type: String

Description: Directory that was read (including a trailing '/')
names

Type: String[:]

Description: File and directory names of a directory in any order

Outputs (2)

orderedNames

Type: String[size(names, 1)]

Description: Names of directories followed by names of files
nDirectories

Type: Integer

Description: The first nDirectories entries in orderedNames are directories
