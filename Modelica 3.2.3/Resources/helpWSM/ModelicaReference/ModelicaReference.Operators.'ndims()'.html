'ndims()'

ndims()

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

Return number of array dimensions

Syntax

ndims(A)

Description

Returns the number of dimensions k of array expression A, with k >= 0.

Examples

Real A[8,4,5];
Integer n = ndims(A);  // = 3
