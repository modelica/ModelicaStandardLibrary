'initial()'

initial()

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

True during initialization

Syntax

initial()

Description

Returns true during the initialization phase and false otherwise.

Examples

  Boolean off;
  Real x;
equation
  off = x < -2 or initial();
