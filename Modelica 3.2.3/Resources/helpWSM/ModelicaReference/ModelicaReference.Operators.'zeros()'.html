'zeros()'

zeros()

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

Returns a zero array.

Syntax

zeros(n1, n2, n3, ...)

Description

Returns the n1 x n2 x n3 x ... Integer array with all elements equal to zero (ni >= 0).
