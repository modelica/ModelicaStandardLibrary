fromReal

Construct Complex from Real

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

This function returns a Complex number defined by real part re and optional imaginary part im (default=0).

Syntax

result = fromReal(re, im)

Inputs (2)

re

Type: Real

Description: Real part of complex number
im

Default Value: 0

Type: Real

Description: Imaginary part of complex number

Outputs (1)

result

Type: Complex

Description: Complex number
