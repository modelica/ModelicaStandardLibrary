Information

The linear admittance connects the complex voltage v with the complex current i by v*G = i. The impedance G_ref is given as complex input signal, representing the resistive and reactive component of the input admittance. The resistive component is modeled temperature dependent, so the real part G = real(Y) is determined from the actual operating temperature and the reference input conductance real(Y_ref).

The variable admittance model has a conditional heat port. A linear temperature dependency of the conductance is taken into account.

Note

A zero crossing of the either the real or imaginary part of the Y_ref signal could cause singularities due to the actual structure of the connected network.
