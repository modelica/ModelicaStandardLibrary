Modelica.Electrical.MultiPhase.Ideal

Multiphase components with idealized behaviour

Information




This package contains analog electrical multiphase components with idealized behaviour,
like thyristor, diode, switch, transformer.





Extends from Modelica.Icons.Library2 (Icon for library where additional icon elements shall be added).
