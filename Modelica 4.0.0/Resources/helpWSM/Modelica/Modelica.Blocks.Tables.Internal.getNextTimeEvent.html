getNextTimeEvent

Return next time event value of 1-dim. table where first column is time

Inputs (2)

Type: ExternalCombiTimeTable

Description: External table object
timeIn

Type: Real

Description: (Scaled) time value

Outputs (1)

nextTimeEvent

Type: Real

Description: (Scaled) next time event in table
