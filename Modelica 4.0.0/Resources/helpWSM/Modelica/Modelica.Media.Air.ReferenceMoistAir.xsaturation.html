xsaturation

Return absolute humidity per unit mass of dry air at saturation as a function of the thermodynamic state record

Syntax

x_sat = xsaturation(state)

Inputs (1)

state

Type: ThermodynamicState

Description: Thermodynamic state record

Outputs (1)

x_sat

Type: MassFraction (kg/kg)

Description: Absolute humidity per unit mass of dry air
