lossConstant_D_zeta

Return the loss constant 8*zeta/(pi^2*D^4)

Syntax

k = lossConstant_D_zeta(D, zeta)

Inputs (2)

D

Type: Diameter (m)

Description: Diameter at port_a or port_b
zeta

Type: Real

Description: Constant pressure loss factor with respect to D (i.e., either port_a or port_b)

Outputs (1)

k

Type: Real

Description: Loss constant (= 8*zeta/(pi^2*D^4))
