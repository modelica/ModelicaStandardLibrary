cp_dT

Specific heat capacity at constant pressure as function of density and temperature

Syntax

cp = cp_dT(d, T, phase, region)

Inputs (4)

d

Type: Density (kg/m³)

Description: Density
T

Type: Temperature (K)

Description: Temperature
phase

Default Value: 0

Type: Integer

Description: 2 for two-phase, 1 for one-phase, 0 if not known
region

Default Value: 0

Type: Integer

Description: If 0, region is unknown, otherwise known and this input

Outputs (1)

cp

Type: SpecificHeatCapacity (J/(kg·K))

Description: Specific heat capacity
