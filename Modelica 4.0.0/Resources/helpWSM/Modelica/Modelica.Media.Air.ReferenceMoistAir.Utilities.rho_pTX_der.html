rho_pTX_der

Derivative of density as a function of pressure p, temperature T and composition X

Syntax

d_der = rho_pTX_der(p, T, X, p_der, T_der, X_der)

Inputs (6)

p

Type: AbsolutePressure (Pa)

Description: Pressure
T

Type: Temperature (K)

Description: Temperature
X

Default Value: Modelica.Media.Air.ReferenceMoistAir.reference_X

Type: MassFraction[:]

Description: Mass fractions
p_der

Type: Real

Description: Derivative of pressure
T_der

Type: Real

Description: Derivative of temperature
X_der

Type: Real[:]

Description: Derivative of mass fractions

Outputs (1)

d_der

Type: Real

Description: Derivative of density
