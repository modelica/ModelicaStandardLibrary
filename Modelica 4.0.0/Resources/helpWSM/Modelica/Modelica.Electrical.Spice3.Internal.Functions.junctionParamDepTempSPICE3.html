junctionParamDepTempSPICE3

Temperature dependency of junction parameters

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

This internal function calculates several temperature dependent junction parameters based on the actual and the nominal temperature.

Syntax

(junctionpot, junctioncap) = junctionParamDepTempSPICE3(phi0, cap0, mcoeff, temp, tnom)

Inputs (5)

phi0

Type: Voltage (V)
cap0

Type: Real
mcoeff

Type: Real
temp

Type: Temperature (K)

Description: Device temperature
tnom

Type: Temperature (K)

Description: Nominal temperature

Outputs (2)

junctionpot

Type: Voltage (V)

Description: Junction potential
junctioncap

Type: Real

Description: Junction capacitance
