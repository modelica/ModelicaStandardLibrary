mos2ModelLineParamsInitEquationsRevised

Initial precalculation

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

This function mos2ModelLineParamsInitEquationsRevised does the initial precalculation of the MOSFET model line parameters for level 2.

Syntax

out_p = mos2ModelLineParamsInitEquationsRevised(in_p, in_m_type)

Inputs (2)

in_p

Type: Mos2ModelLineParams

Description: Input record model line parameters for MOS2
in_m_type

Type: Integer

Description: Type of MOS transistor

Outputs (1)

out_p

Type: Mos2ModelLineParams

Description: Input record model line parameters for MOS2
