VoidFraction

Calculation of (cross sectional) void fraction for two phase flow

Syntax

(epsilon, slipRatio) = VoidFraction(voidFractionApproach, crossSectionalAveraged, rho_g, rho_l, x_flow)

Inputs (5)

voidFractionApproach

Default Value: Dissipation.Utilities.Types.VoidFractionApproach.Homogeneous

Type: VoidFractionApproach

Description: Choice of void fraction approach
crossSectionalAveraged

Default Value: true

Type: Boolean

Description: = true, if cross sectional averaged void fraction, otherwise volumetric
rho_g

Type: Density (kg/m³)

Description: Density of gaseous phase
rho_l

Type: Density (kg/m³)

Description: Density of liquid phase
x_flow

Default Value: 0

Type: Real

Description: Mass flow rate quality

Outputs (2)

epsilon

Type: Real

Description: Void fraction
slipRatio

Type: Real

Description: Slip ratio
