velocityOfSound_ph

Syntax

v_sound = velocityOfSound_ph(p, h, phase, region)

Inputs (4)

p

Type: Pressure (Pa)

Description: Pressure
h

Type: SpecificEnthalpy (J/kg)

Description: Specific enthalpy
phase

Default Value: 0

Type: Integer

Description: 2 for two-phase, 1 for one-phase, 0 if not known
region

Default Value: 0

Type: Integer

Description: If 0, region is unknown, otherwise known and this input

Outputs (1)

v_sound

Type: Velocity (m/s)

Description: Speed of sound
