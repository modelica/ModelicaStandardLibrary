gibbsToProps_dT

Calculate property record for density and temperature as states from dimensionless Gibbs function

Syntax

pro = gibbsToProps_dT(g)

Inputs (1)

g

Type: GibbsDerivs

Description: Dimensionless derivatives of Gibbs function

Outputs (1)

pro

Type: ThermoProperties_dT

Description: Property record for density and temperature as dynamic states
