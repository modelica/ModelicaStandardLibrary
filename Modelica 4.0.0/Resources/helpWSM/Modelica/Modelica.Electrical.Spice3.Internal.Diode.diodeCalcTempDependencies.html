diodeCalcTempDependencies

Temperature dependency calculation

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

In this function for the diode model temperature dependencies are calculated using temperature treating functions from the equation package.

Syntax

out_c = diodeCalcTempDependencies(in_p, in_dp, in_m, in_v)

Inputs (4)

in_p

Type: DiodeModelLineParams

Description: Input record diode model line parameters
in_dp

Type: DiodeParams

Description: Input record diode parameters
in_m

Type: Model

Description: Input record Model
in_v

Type: DiodeVariables

Description: Input record diode variables

Outputs (1)

out_c

Type: DiodeCalc

Description: Output record with calculated values
