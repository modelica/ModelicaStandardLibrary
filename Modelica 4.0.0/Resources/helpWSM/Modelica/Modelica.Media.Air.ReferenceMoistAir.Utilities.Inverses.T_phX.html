T_phX

Return temperature as a function of pressure, specific enthalpy and mass fractions

Syntax

T = T_phX(p, h, X)

Inputs (3)

p

Type: AbsolutePressure (Pa)

Description: Pressure
h

Type: SpecificEnthalpy (J/kg)

Description: Specific enthalpy
X

Default Value: Modelica.Media.Air.ReferenceMoistAir.reference_X

Type: MassFraction[:]

Description: Mass fractions

Outputs (1)

T

Type: Temperature (K)

Description: Temperature
