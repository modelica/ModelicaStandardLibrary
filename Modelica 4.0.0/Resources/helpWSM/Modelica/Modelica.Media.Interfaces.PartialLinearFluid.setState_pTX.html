setState_pTX

Set the thermodynamic state record from p and T (X not needed)

Syntax

state = setState_pTX(p, T, X)

Inputs (3)

p

Type: AbsolutePressure (Pa)

Description: Pressure
T

Type: Temperature (K)

Description: Temperature
X

Default Value: reference_X

Type: MassFraction[:] (kg/kg)

Description: Mass fractions

Outputs (1)

state

Type: ThermodynamicState

Description: Thermodynamic state record
