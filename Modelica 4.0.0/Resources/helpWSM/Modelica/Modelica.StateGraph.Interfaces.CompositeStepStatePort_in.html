CompositeStepStatePort_in

Communication port between a CompositeStep and the ordinary steps within the CompositeStep (suspend/resume are inputs)

Inputs (2)

suspend

Type: Boolean

Description: = true, if suspend transition of CompositeStep fires
resume

Type: Boolean

Description: = true, if resume transition of CompositeStep fires

Used in Components (1)
