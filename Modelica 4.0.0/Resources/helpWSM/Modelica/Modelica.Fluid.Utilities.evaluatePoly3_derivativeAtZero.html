evaluatePoly3_derivativeAtZero

Evaluate polynomial of order 3 that passes the origin with a predefined derivative

Syntax

y = evaluatePoly3_derivativeAtZero(x, x1, y1, y1d, y0d)

Inputs (5)

x

Type: Real

Description: Value for which polynomial shall be evaluated
x1

Type: Real

Description: Abscissa value
y1

Type: Real

Description: y1=f(x1)
y1d

Type: Real

Description: First derivative at y1
y0d

Type: Real

Description: First derivative at f(x=0)

Outputs (1)

y

Type: Real
