junctionVoltage23SPICE3

Junction Voltage

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

This internal function calculates the junction voltage based on the actual temperature, voltage and saturation current.

Syntax

v23 = junctionVoltage23SPICE3(vb, ivb, satcur, temp, ncoeff)

Inputs (5)

vb

Type: Voltage (V)
ivb

Type: Current (A)
satcur

Type: Current (A)

Description: Saturation current
temp

Type: Temperature (K)

Description: Device temperature
ncoeff

Type: Real

Outputs (1)

v23

Type: Voltage (V)

Description: Output value
