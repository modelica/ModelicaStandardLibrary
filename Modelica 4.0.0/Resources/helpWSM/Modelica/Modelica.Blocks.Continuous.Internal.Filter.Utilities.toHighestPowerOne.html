toHighestPowerOne

Transform filter to form with highest power of s equal 1

Syntax

(cr, c0, c1) = toHighestPowerOne(den1, den2)

Inputs (2)

den1

Type: Real[:]

Description: [s] coefficients of polynomials (den1[i]*s + 1)
den2

Type: Real[:,2]

Description: [s^2, s] coefficients of polynomials (den2[i,1]*s^2 + den2[i,2]*s + 1)

Outputs (3)

cr

Type: Real[size(den1, 1)]

Description: [s^0] coefficients of polynomials cr[i]*(s+1/cr[i])
c0

Type: Real[size(den2, 1)]

Description: [s^0] coefficients of polynomials (s^2 + (den2[i,2]/den2[i,1])*s + (1/den2[i,1]))
c1

Type: Real[size(den2, 1)]

Description: [s^1] coefficients of polynomials (s^2 + (den2[i,2]/den2[i,1])*s + (1/den2[i,1]))
