symmetricOrientation

Orientations of the resulting fundamental wave field phasors

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

This function determines the orientation angles of the symmetrical winding with m phases.

See also

User's guide on symmetrical components and orientation.

Syntax

orientation = symmetricOrientation(m)

Inputs (1)

m

Type: Integer

Description: Number of phases

Outputs (1)

orientation

Type: Angle[m] (rad)

Description: Orientation of the resulting fundamental wave field phasors
