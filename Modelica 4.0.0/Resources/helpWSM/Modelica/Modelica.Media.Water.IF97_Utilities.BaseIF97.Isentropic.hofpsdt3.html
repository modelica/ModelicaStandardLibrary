hofpsdt3

Isentropic specific enthalpy in region 3 h(p,s) with given good guess in d and T

Syntax

h = hofpsdt3(p, s, dguess, Tguess, delp, dels)

Inputs (6)

p

Type: Pressure (Pa)

Description: Pressure
s

Type: SpecificEntropy (J/(kg·K))

Description: Specific entropy
dguess

Type: Density (kg/m³)

Description: Good guess density, e.g., from adjacent volume
Tguess

Type: Temperature (K)

Description: Good guess temperature, e.g., from adjacent volume
delp

Default Value: IterationData.DELP

Type: Pressure (Pa)

Description: Relative error in p
dels

Default Value: IterationData.DELS

Type: SpecificEntropy (J/(kg·K))

Description: Relative error in s

Outputs (1)

h

Type: SpecificEnthalpy (J/kg)

Description: Specific enthalpy
