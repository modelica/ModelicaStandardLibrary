mosCalcDEVqmeyer

Calculation of Meyer capacities

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

This function mosCalcDEVqmeyer calculates the Meyer capacities and charge for the Meyer model.

Syntax

out_qm = mosCalcDEVqmeyer(vgs, vgd, vgb, in_c)

Inputs (4)

vgs

Type: Voltage (V)
vgd

Type: Voltage (V)
vgb

Type: Voltage (V)
in_c

Type: MosCalc

Description: Input variable set

Outputs (1)

out_qm

Type: DEVqmeyer

Description: Qmeyer values
