SlipRatio

Calculation of (analytical/empirical) slip ratio

Syntax

SR = SlipRatio(voidFractionApproach, rho_g, rho_l, x_flow)

Inputs (4)

voidFractionApproach

Default Value: Dissipation.Utilities.Types.VoidFractionApproach.Homogeneous

Type: VoidFractionApproach

Description: Choice of void fraction approach
rho_g

Type: Density (kg/m³)

Description: Density of gaseous phase
rho_l

Type: Density (kg/m³)

Description: Density of liquid phase
x_flow

Default Value: 0

Type: Real

Description: Mass flow rate quality

Outputs (1)

SR

Type: Real

Description: Slip ratio
