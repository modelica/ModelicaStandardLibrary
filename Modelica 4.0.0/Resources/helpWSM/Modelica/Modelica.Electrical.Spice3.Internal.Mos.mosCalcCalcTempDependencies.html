mosCalcCalcTempDependencies

Precalculation relating to temperature

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

This function mosCalcCalcTempDependencies does precalculation relating to the temperature (level 1).

Syntax

out_c = mosCalcCalcTempDependencies(in_p, in_C, in_vp, in_m, in_c, in_m_type)

Inputs (6)

in_p

Type: Mos1ModelLineParams

Description: Input record model line parameters for MOS1
in_C

Type: SpiceConstants

Description: Input record SPICE constants
in_vp

Type: MosModelLineVariables

Description: Input record model line variables
in_m

Type: Mosfet

Description: Input record MOSFET parameters
in_c

Type: Mos1Calc

Description: Input record Mos1Calc
in_m_type

Type: Integer

Description: Type of MOS transistor

Outputs (1)

out_c

Type: Mos1Calc

Description: Output record with calculated values
