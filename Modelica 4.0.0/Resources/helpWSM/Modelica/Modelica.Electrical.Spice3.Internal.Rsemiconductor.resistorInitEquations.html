resistorInitEquations

Initial calculation

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

In this function some initial calculations for the resistor model are done, especially concerning the handling of the breakthrough voltage.

Syntax

out = resistorInitEquations(in_p, in_p2)

Inputs (2)

in_p

Type: ResistorParams

Description: Input record with resistor parameters
in_p2

Type: ResistorModelLineParams

Description: Input record with resistor model line parameters

Outputs (1)

out

Type: ResistorVariables

Description: Output record with resistor variables
