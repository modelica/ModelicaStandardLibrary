specificEnthalpyOfT

Return specific enthalpy from pressure and temperature, taking the flag enthalpyOfT into account

Syntax

h = specificEnthalpyOfT(p, T, densityOfT)

Inputs (3)

p

Type: AbsolutePressure (Pa)

Description: Pressure
T

Type: Temperature (K)

Description: Temperature
densityOfT

Default Value: false

Type: Boolean

Description: Include or neglect density derivative dependence of enthalpy

Outputs (1)

h

Type: SpecificEnthalpy (J/kg)

Description: Specific enthalpy
