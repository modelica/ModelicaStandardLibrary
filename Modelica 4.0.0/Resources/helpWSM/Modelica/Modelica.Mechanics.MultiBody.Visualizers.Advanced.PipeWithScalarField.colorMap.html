colorMap

Function defining the color map

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

This replaceable function defines a particular color map.

Syntax

colorMap = colorMap(n_colors)

Inputs (1)

n_colors

Default Value: 64

Type: Integer

Description: Number of colors in the color map

Outputs (1)

colorMap

Type: Real[n_colors,3]

Description: Color map to map a scalar to a color
