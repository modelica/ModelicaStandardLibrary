calculateGateCap

Gate capacitance calculation

Syntax

(capgs, chargegs, capgd, chargegd) = calculateGateCap(in_f, vgs, vgd)

Inputs (3)

in_f

Type: Fet
vgs

Type: Voltage (V)

Description: Input voltage gate source
vgd

Type: Voltage (V)

Description: Input voltage gate drain

Outputs (4)

capgs

Type: Capacitance (F)

Description: Output capacitance gate source
chargegs

Type: Charge (C)

Description: Output charge gate source
capgd

Type: Capacitance (F)

Description: Output capacitance gate drain
chargegd

Type: Charge (C)

Description: Output charge gate drain
