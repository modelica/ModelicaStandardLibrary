helmholtzToProps_pT

Calculate property record for pressure and temperature as states from dimensionless Helmholtz function

Syntax

pro = helmholtzToProps_pT(f)

Inputs (1)

f

Type: HelmholtzDerivs

Description: Dimensionless derivatives of Helmholtz function

Outputs (1)

pro

Type: ThermoProperties_pT

Description: Property record for pressure and temperature as dynamic states
