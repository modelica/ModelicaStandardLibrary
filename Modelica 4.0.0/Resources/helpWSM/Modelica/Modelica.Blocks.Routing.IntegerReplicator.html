IntegerReplicator

Integer signal replicator

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

This block replicates the Integer input signal to an array of nout identical Integer output signals.

Parameters (1)

nout

Value: 1

Type: Integer

Description: Number of outputs

Connectors (2)

u

Type: IntegerInput

Description: Connector of Integer input signal
y

Type: IntegerOutput[nout]

Description: Connector of Integer output signals
