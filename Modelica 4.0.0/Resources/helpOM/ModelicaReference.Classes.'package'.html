Information

Define specialized class package

Examples

package Library
  constant Real k = 0.1;

  type X = Real(min=0);

  model A
    ...
  end A;

  model B
    ...
  end B;
end Library;

Description

May only contain declarations of classes and constants. Enhanced to allow import of elements of packages.
