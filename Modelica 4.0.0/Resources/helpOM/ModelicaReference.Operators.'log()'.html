Natural (base e) logarithm

Syntax

log(u)

Description

Returns the base e logarithm of u, with u > 0. Argument u needs to be an Integer or Real expression.

The natural logarithm can also be accessed as Modelica.Math.log.

Examples

log(1)
 = 0
